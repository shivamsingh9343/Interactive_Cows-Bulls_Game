# 🐮🐂 Cows and Bulls Game (Python – Google Colab)

An interactive implementation of the classic **Cows and Bulls puzzle game** using Python and **ipywidgets** in Google Colab.  
Players must guess a randomly generated 3‑digit number with unique digits. After each guess, the system provides feedback in terms of:

- **Bulls 🐂** → Correct digit in the correct position  
- **Cows 🐮** → Correct digit but in the wrong position  

The game continues until the player guesses correctly or runs out of 10 tries.

---

## ✨ Features
- **[Random number generation](ca://s?q=Random_number_generation_in_Python)** with unique digits

- **[Interactive widgets](ca://s?q=ipywidgets_example_in_Jupyter)** (input box, buttons, reset option)  
- **[Guess history tracking](ca://s?q=Guess_history_tracking_in_Cows_and_Bulls)** in a structured table  
- **[Win/Lose messages](ca://s?q=Win_and_Lose_messages_in_Cows_and_Bulls)** with images and feedback  
- **[Reset functionality](ca://s?q=Reset_button_in_Cows_and_Bulls)** to play multiple rounds  

---

## 🚀 Setup & Usage
1. Open the notebook in **Google Colab**.  
2. Install required libraries (see `requirements.txt`).  
3. Run all cells to start the game.  
4. Enter guesses in the input box and click **Check Guess**.  
5. Use **Play Again** to reset the game after win/lose.
